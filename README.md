### <p align="center">🤔 Purpose 🤔</p>
**With GreenLog you can extract assets such as :**

    • Images
    • Icons
    • Sound effects
    • Music

**From any <a href="https://www.roblox.com/home">Roblox</a> experience!**

-----

### <p align="center">📜 Requirements 📜</p>
**To utilize GreenLog you need :**

    • Windows 10 or 11
    • Python 3.11+
    • An active internet connection
    • Roblox

**You will also need to install a few Python packages with this CMD command :**

```pip install --upgrade -r requirements.txt```
